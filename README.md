- Taro with React

- store、network - works out-of-box

- [rematch](https://github.com/rematch/rematch) - easy Redux 

- TypeScript

- [tailwindcss](https://www.tailwindcss.com/)

## Usage

1. Clone this repository
2. npm install 
3. npm run dev:weapp
